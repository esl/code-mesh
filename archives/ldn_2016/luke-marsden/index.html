The Sock Shop: Microservices from dev to prod  with Docker and Kubernetes

In this talk, we’ll show off our sample reference microservices app, a socks shop! We’ll look at how we developed it and why, and then deploy it locally and to a production orchestration framework. In both cases we'll use Weave Scope to see what it’s doing. Finally, I'll dig into how Weave Net uses Gossip and CRDTs to ensure that the container network can continue operating on both sides of a network partition.

Talk Objectives: 

  • Understand practically how to deploy a real microservices app locally to Docker in dev and in production to Kubernetes.

Target Audience: 

  • Developers & devops engineers.
